# Boris Cherny — Claude Code 15 大專業技巧

> 來源：[bnext.com.tw](https://www.bnext.com.tw/article/90524/claude-code-boris-cherny-15-pro-tips-auto-dev-workflow)

1. 手機寫程式 — 隨時隨地用行動裝置下指令
2. 跨裝置交接 — 桌機與手機無縫切換工作階段
3. `/loop` 自動循環 — 讓 Claude 持續執行重複任務
4. Hooks 生命週期 — PreToolUse / PostToolUse / Stop 自動化鉤子
5. Cowork Dispatch — 從手機派發任務到桌機執行
6. Chrome 擴充功能 — 瀏覽器內直接呼叫 Claude Code
7. Web Server 測試 — 啟動本地伺服器讓 Claude 自行驗證
8. Session Forking（`/branch`）— 分叉對話探索不同解法
9. `/btw` 背景備忘 — 不中斷流程地傳遞上下文
10. Git Worktrees — 多工作區並行開發不衝突
11. `/batch` 批次執行 — 同時處理多個獨立任務
12. `--bare` 模式 — 無互動的純指令輸出
13. `--add-dir` 多目錄 — 跨專案資料夾協作
14. `--agents` 多代理人 — 平行啟動多個 subagent
15. `/voice` 語音輸入 — 用說的下指令
