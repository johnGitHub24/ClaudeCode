# Claude Code 快速學習指南

互動式網頁教材，幫助你快速掌握 Claude Code CLI 的核心功能與進階技巧。

## 專案說明

本專案是一個純前端單頁應用，無需任何框架或建置工具，用瀏覽器直接開啟即可使用。內容涵蓋從安裝入門到進階實戰的完整學習路徑。

## 主要功能

- **安裝與設定** — 環境準備、API 金鑰、基本設定
- **核心指令** — 常用 slash commands、快捷鍵、CLI flags
- **模組化學習** — 6 大學習模組循序漸進
- **15 大專業技巧** — Boris Cherny 整理的進階工作流技巧
- **Cowork 遠端控制** — Remote Control、Dispatch、Channels 完整說明
- **自訂 Subagents** — 建立與管理專屬 AI 代理人
- **資源參考區** — 官方文件、影片教學、社群連結

## 使用方式

```bash
git clone https://github.com/johnGitHub24/ClaudeCode.git
cd ClaudeCode
# 用瀏覽器開啟 index.html 即可
open index.html   # macOS
start index.html  # Windows
```

## 技術棧

- 純 HTML5 / CSS3 / JavaScript（ES6+）
- 無任何外部依賴或框架
- 單一檔案 `index.html`，離線可用
